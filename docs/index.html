Vision Trigger Platform

Vision Trigger Platform

Webカメラの映像をAIがリアルタイムにテキスト化し、 あらかじめ登録した状況が発生したらOSC/Socket.IOで外部アプリに通知するプラットフォーム。 ROI（関心領域）を自由に設定でき、領域ごとの状況監視が可能です。

An AI-powered platform that converts webcam footage to text in real-time using BLIP, and triggers OSC/Socket.IO notifications when registered situations are detected. Supports multiple ROI (Region of Interest) zones for independent monitoring.

Features

Features

🧠

AIキャプション生成

AI Captioning

BLIPモデルがカメラ映像をリアルタイムに英語テキストに変換。Apple Silicon（MPS）で高速推論。

BLIP model converts camera frames to English text in real-time. Fast inference on Apple Silicon (MPS).

🎯

テキスト類似度トリガー

Text Similarity Triggers

監視したい状況をテキストで登録。Sentence-Transformerでキャプションとの意味的類似度を計算し、閾値を超えたら発火。

Register situations as text. Sentence-Transformer computes semantic similarity with captions and fires when threshold is exceeded.

🔲

ROI（関心領域）

ROI (Region of Interest)

カメラ映像上に複数のROIを描画。各領域を独立してキャプション生成・トリガー判定。

Draw multiple ROIs on camera feed. Each region gets independent captioning and trigger evaluation.

📡

OSC + Socket.IO出力

OSC + Socket.IO Output

トリガー発火時にOSCとSocket.IOで外部アプリに通知。Pure Data、TouchDesigner、Max等と連携。

Notify external apps via OSC and Socket.IO on trigger. Works with Pure Data, TouchDesigner, Max, etc.

📊

リアルタイムモニター

Real-time Monitor

類似度バー、OSCモニター、キャプションログをダッシュボードでリアルタイム表示。閾値はドラッグで調整可能。

Similarity bars, OSC monitor, and caption log displayed in real-time dashboard. Threshold adjustable by drag.

⚡

MPS高速推論

MPS Acceleration

PyTorch MPSバックエンドでApple Silicon GPUを活用。1フレームあたり約0.4秒で推論。

Leverages Apple Silicon GPU via PyTorch MPS backend. ~0.4s inference per frame.
